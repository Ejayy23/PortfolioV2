I developed a restaurant website as a personal project using HTML, CSS, and JavaScript for the front end, and Django for the back end. Inventory Management System

Inventory Management System I developed using C# with MySQL in Visual Studio is a solution for businesses to efficiently manage their stock, track product movements, and streamline operations. By integrating C#, the system provides a user-friendly interface for adding, updating, and deleting inventory records, while MySQL ensures secure and scalable database management for storing essential data such as product details, stock levels, and transaction history.

Membership Management System

A Membership Management System built using PHP and MySQL is designed to efficiently handle user registrations, memberships, and administrative tasks. This system allows users to sign up, log in, and manage their profiles while providing administrators with tools to oversee member data, track subscriptions, and generate reports. The backend, powered by MySQL, securely stores user credentials, membership details, and transaction histories, ensuring data integrity and efficient retrieval.
